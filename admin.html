Sign in with a GitHub access token to write posts and update the site. Changes are committed straight to the repository and go live once GitHub Pages rebuilds.

How do I get a token?
  1. On GitHub go to Settings → Developer settings → Personal access tokens → Fine-grained tokens.
  2. Create a token scoped to only the annay-de/annay.de repository.
  3. Under repository permissions, give Contents read and write access. Nothing else is needed.
  4. Paste the token above. It is only stored in this browser and only ever sent to api.github.com.
